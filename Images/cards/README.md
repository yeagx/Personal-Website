# Clash Royale Card Images

This folder contains images for the Clash Royale cards displayed in the portfolio.

## Required Images

To display the cards properly, you need to add the following images:

1. `goblin-barrel.png` - Goblin Barrel card
2. `princess.png` - Princess card  
3. `knight.png` - Knight card
4. `ice-spirit.png` - Ice Spirit card
5. `goblin-gang.png` - Goblin Gang card
6. `inferno-tower.png` - Inferno Tower card
7. `rocket.png` - Rocket card
8. `log.png` - The Log card

## Image Requirements

- **Size**: Recommended 80x100 pixels (or similar aspect ratio)
- **Format**: PNG with transparent background preferred
- **Quality**: High quality, clear images

## Where to Get Images

You can find official Clash Royale card images from:
- Official Clash Royale website
- Clash Royale Wiki
- Or use placeholder images for now

## Fallback

If images are not found, the website will display colored placeholders with card abbreviations.
